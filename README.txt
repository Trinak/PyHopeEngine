PyHopeEngine is a game engine written in python and based on pygame. It is based on the work of Mike McShaffry and David "Rez" Graham and their book  Game Coding Complete (4th edition). The code for that engine (written in C++) can be found at https://code.google.com/p/gamecode4/


Required third party libraries:
--glyph
--Mastermind
--PGU (Python Game Utilities)
--pygame
--pymunk
--pyganim
--pycallgraph (only to use the profiling utility)
